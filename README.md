인프라 운영 지능화
===============
기능 요건
---------
1. BYOD(Bring Your Own Device): 개인 소유 기기를 이용한 운영 업무 지원
2. 운영자를 위한 쌍방향의 인터렉티브(Interactive)한 기능 제공: 쳇봇
3. 관련자를 위한 상황 공유 기능 제공: 단체 알림
4. 이벤트 감지: 임계치 설정/감지(Rule 기반), Alert Manager
5. 시계열 기반 데이터 저장소
6. 데이터 시각화
